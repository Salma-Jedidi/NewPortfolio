Gestion des Rendez-vous Médicaux

Medical Appointment Management

Lien GitHub App Mobile

Le projet RDVMed est une application web et mobile permettant aux patients de prendre des rendez-vous médicaux facilement et aux professionnels de santé de gérer leurs agendas efficacement.
Ce projet utilise Spring Boot pour le backend et Angular pour le frontend, avec une architecture basée sur le modèle MVC.


Technologies utilisées :
  1. Frontend :
    • Angular : Développement de l'interface utilisateur dynamique et réactive, gestion des composants et des services pour améliorer l'expérience utilisateur.
    • HTML5 / CSS3 / JavaScript : Conception et mise en page de l'application avec des fonctionnalités front-end personnalisées.
  2. Backend :
    • Spring Boot : Création des services RESTful pour la gestion des données des utilisateurs et des rendez-vous.
    • Hibernate : Gestion de la persistance des données avec une base relationnelle (MySQL).
  3. Architecture :
    • MVC : Mise en œuvre de la séparation des responsabilités pour un développement organisé et évolutif.
  4. Sécurité :
    • JWT : Sécurisation de l'accès aux services via l'authentification par tokens.
  5. Reconnaissance Faciale :
    • Utilisation de la bibliothèque face_recognition pour identifier les utilisateurs à partir de leurs photos.
    • Fonctionnalités principales :
      • Enregistrement : Analyse et stockage de l’encodage facial lors de l’inscription d’un utilisateur.
      • Authentification : Comparaison en temps réel de l’encodage facial pour la connexion sécurisée.
  6. Chatbot :
    • Basé sur le modèle DialoGPT de Microsoft pour une interaction conversationnelle avancée.
    • Fonctionnalités principales :
      • Réception des messages des utilisateurs via des requêtes POST.
      • Génération de réponses pertinentes en utilisant l’IA.
      • Gestion de l’historique des conversations pour un contexte personnalisé.

Conclusion :

Ce projet m’a permis de développer mes compétences en développement full-stack, en gestion de projet MVC, et en implémentation de sécurité. RDVMed offre une solution complète et évolutive pour la gestion des rendez-vous médicaux.

The RDVMed project is a web and mobile application allowing patients to book medical appointments easily and healthcare professionals to manage their schedules effectively.
This project uses Spring Boot for the backend and Angular for the frontend, following the MVC model.


Technologies used:
  1. Frontend:
    • Angular: Development of a dynamic and interactive user interface with optimized components and services.
    • HTML5 / CSS3 / JavaScript: Design and layout of the application with custom front-end features.
  2. Backend:
    • Spring Boot: Creation of RESTful services for user and appointment data management.
    • Hibernate: Data persistence management with a relational database (MySQL).
  3. Architecture:
    • MVC: Clear separation of concerns for organized and scalable development.
  4. Security:
    • JWT: Securing access to services through token-based authentication.
  5. Facial Recognition:
    • Using the face_recognition library to identify users based on their photos.
    • Main features:
      • Registration: Analyzing and storing facial encoding during user registration.
      • Authentication: Real-time comparison of facial encoding for secure login.
  6. Chatbot:
    • Based on Microsoft's DialoGPT model for advanced conversational interaction.
    • Main features:
      • Receiving user messages via POST requests.
      • Generating relevant responses using AI.
      • Managing conversation history for personalized context.

Conclusion:

This project enhanced my full-stack development skills, MVC project management, and security implementation. RDVMed provides a complete and scalable solution for managing medical appointments.
