Time Master - Time and Activity Management

Lien GitHub

Le projet Time Master est une application web et mobile conçue pour optimiser la gestion du temps et des activités quotidiennes. Il permet aux utilisateurs de suivre et organiser leurs tâches en fonction de priorités et de délais.

Dans ce projet, j’ai utilisé Angular pour le frontend et Spring Boot pour le backend. Le système repose sur une architecture basée sur des microservices, offrant ainsi une grande flexibilité et évolutivité.


Technologies utilisées:
  1. Frontend :
    • Angular : Utilisé pour créer une interface dynamique et interactive. Gestion des vues, des composants et des services pour optimiser l'expérience utilisateur.
    • HTML5 / CSS3 / JavaScript : Structuration, mise en page et interactivité de l’application avec des fonctionnalités front-end personnalisées.
  2. Backend :
    • Spring Boot : Framework Java utilisé pour développer des services RESTful efficaces pour la gestion des données des utilisateurs et des tâches.
    • Hibernate : Utilisé pour la gestion de la persistance des données avec une base de données relationnelle (MySQL ou PostgreSQL).
  3. Architecture :
    • MVC : L'architecture a permis une séparation claire des préoccupations et un développement efficace. L'utilisation de Spring Boot pour le backend et d'Angular pour le frontend a apporté flexibilité et évolutivité pour les améliorations futures.
  4. Base de données :
    • MySQL : Système de gestion de base de données relationnelle utilisé pour stocker les données des utilisateurs et des activités.
  5. Authentification et Sécurité :
    • JWT : Implémentation des JSON Web Tokens pour sécuriser l’accès aux services backend et assurer une gestion des sessions sécurisée.

Conclusion :

Ce projet m’a permis de renforcer mes compétences en gestion de projet full-stack, en architecture de microservices, ainsi qu’en développement sécurisé avec Spring Boot et Angular. Le projet offre une solution robuste et scalable pour la gestion du temps et des tâches, et il peut être adapté à divers cas d'utilisation dans différents domaines professionnels.

The Time Master project is a web and mobile application designed to optimize the management of time and daily activities. It allows users to track and organize their tasks according to priorities and deadlines.

In this project, I used Angular for the frontend and Spring Boot for the backend. The system relies on a microservices architecture, providing flexibility and scalability.


Technologies used:
  1. Frontend:
    • Angular: Used to create a dynamic and interactive user interface. Management of views, components, and services to enhance the user experience.
    • HTML5 / CSS3 / JavaScript: Structuring, styling, and adding interactivity to the application with custom frontend features.
  2. Backend:
    • Spring Boot: Java framework used to develop RESTful services for managing user data and tasks.
    • Hibernate: Used to manage data persistence with a relational database (MySQL or PostgreSQL).
  3. Architecture:
    • MVC: It allowed clear separation of concerns and efficient development. Using Spring Boot for the backend and Angular for the frontend provided flexibility and scalability for future enhancements.
  4. Database:
    • MySQL: Relational database management system used to store user and activity data.
  5. Authentication and Security:
    • JWT: Implemented JSON Web Tokens to secure access to backend services and manage secure sessions.

Conclusion:

This project helped enhance my full-stack project management, microservices architecture, and secure development skills with Spring Boot and Angular. It offers a robust and scalable solution for time and task management, adaptable to various professional fields.
