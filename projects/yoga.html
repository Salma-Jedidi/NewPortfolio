Yoga Movement Prediction

Lien GitHub

Le projet Prédiction des mouvements de yoga vise à développer un modèle de machine learning capable de prédire les positions et mouvements effectués pendant une séance de yoga. À travers ce projet, j'ai appliqué des compétences en traitement des données de capteurs, en analyse des mouvements humains et en construction de modèles prédictifs.
En utilisant des algorithmes comme le Random Forest et la régression logistique, j'ai optimisé les performances du modèle pour prédire avec précision les mouvements. Ce projet a renforcé mes compétences en data science et en programmation Python.


Techniques utilisées:
  1. Manipulation et nettoyage des données :
    • Pandas : Pour gérer et traiter les données collectées par des capteurs de mouvement et des caméras.
    • NumPy : Pour les calculs numériques et la gestion des données multidimensionnelles.
  2. Analyse des mouvements humains :
    • OpenPose / MediaPipe : Pour l'extraction des points de repère du corps humain à partir de vidéos ou de flux de capteurs.
  3. Apprentissage automatique :
    • Scikit-learn : Pour implémenter des algorithmes comme Random Forest et Régression Logistique, et pour évaluer les modèles de prédiction.
    • TensorFlow / Keras : Pour développer des réseaux de neurones profonds, si nécessaire pour des prédictions plus complexes.
  4. Environnement de programmation :
    • Python : Le langage de programmation principal pour le traitement des données et l'implémentation des modèles.
  5. Développement et déploiement :
    • Jupyter Notebook : Pour l'analyse exploratoire des données et le développement des modèles dans un environnement interactif.
    • VS Code : Pour écrire et maintenir le code.
  6. Évaluation et optimisation :
    • Scikit-learn Metrics : Pour évaluer la performance des modèles avec des métriques comme la précision, le rappel et le F1-score.
    • GridSearchCV : Pour optimiser les hyperparamètres des modèles.
  7. Sources de données et stockage :
    • Capteurs de mouvement : Données collectées via des capteurs ou des vidéos enregistrées de séances de yoga.
    • Sources internes : Données collectées manuellement ou générées pour entraîner le modèle.

Conclusion:

Le projet a permis de prédire avec précision les mouvements de yoga en utilisant des données de capteurs. Les facteurs influents incluent l'alignement corporel et la fluidité des mouvements.

  4. Programming Environment:
    • Python: The main programming language used for data processing and model implementation.

  7. Data Sources and Storage:
    • Motion sensors: Data collected via motion sensors or recorded video from yoga sessions.
