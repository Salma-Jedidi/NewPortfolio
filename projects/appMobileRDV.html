Application mobile de Gestion de rendez-vous médicaux

Gestion des Rendez-vous Médicaux

Medical_Appointment_Management



Le projet RDVMed est une application mobile permettant aux patients de prendre des rendez-vous médicaux facilement et aux professionnels de santé de gérer leurs agendas efficacement.
Ce projet utilise Firebase pour le backend et FlutterFlow pour le frontend.


Translate

The RDVMed project is a mobile application allowing patients to book medical appointments easily and healthcare professionals to manage their schedules effectively.
This project uses Firebase for the backend and FlutterFlow for the frontend.
